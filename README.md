# Astrophysical Calculations in Python

This project implements numerical calculations for astrophysical and orbital mechanics problems using Python.

## Topics Covered

- Orbital mechanics
- Satellite motion
- Gravitational calculations
- Stellar and planetary physics

## Methods

The notebook applies Newtonian gravitational equations to estimate orbital properties and astrophysical parameters.

Python  
NumPy  
Matplotlib
